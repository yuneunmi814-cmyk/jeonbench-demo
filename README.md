# 우리가게 에너지벤치 — 웹 데모

소상공인이 전기·가스 고지서를 올리면 **한국가스공사 용도별 요금 공공데이터**로 동종업계 대비 과납 여부를 진단하고 절감 처방을 주는 모바일 웹.
산업부 제14회 공공데이터 공모전 **제품·서비스 개발 부문** 시제품.

**라이브:** https://yuneunmi814-cmyk.github.io/jeonbench-demo/

## 파일
- `index.html` / `style.css` — 모바일 UI (3화면: 고지서 인식 → 벤치마크 → 처방)
- `data.js` — 가스 용도별 단가(한국가스공사 도매 기반)·전기 단가(한전) + 업종별 동종 분포 + 샘플 3종
- `app.js` — 벤치마크 분위 계산·절감 처방 엔진(가스 용도 적정성 / 전기 계약 최적화)

## 실행
```bash
cd 전기벤치_웹데모
python3 -m http.server 8777   # http://localhost:8777
```
샘플: **한식집(가스·업무난방용 오계약 → 상위 22%·연 87만 절감)** / 고깃집(적정사례) / 카페(전기·상위 12%).

## 공공데이터
- **필수**: 한국가스공사_천연가스요금정보(용도별 도매요금) → `data.js > GAS_TARIFF` 의 base
- 기타: 한국전력공사 용도업종별·계약종별 전력사용량 (전기 벤치마크)
- 실서비스: `PEER` 데모 분포 → 한국가스공사 실데이터로 대체
