# 우리가게 전기벤치 — 웹 데모

소상공인이 전기 고지서를 올리면 **한전 공공데이터**로 동종업계 대비 과납 여부를 진단하고 절감 처방을 주는 모바일 웹.
산업부 제14회 공공데이터 공모전 **제품·서비스 개발 부문** 시제품.

## 파일
- `index.html` / `style.css` — 모바일 UI (3화면: 고지서 인식 → 벤치마크 → 처방)
- `data.js` — 한전 요금표(실측) + 업종별 동종 분포(데모) + 샘플 고지서 3종
- `app.js` — 벤치마크 분위 계산·절감 처방 엔진·(선택) Gemini Vision OCR

## 실행
```bash
cd 전기벤치_웹데모
python3 -m http.server 8777
# 브라우저: http://localhost:8777
```
샘플 버튼(카페/한식집/제과점) 클릭 → 즉시 진단. 카페 = 상위 12%·연 156만 과납·연 118만 절감.

## 실제 OCR 연동(선택)
헤더 ⚙ → Gemini API 키 입력(브라우저 localStorage에만 저장) → 고지서 업로드 시 `gemini-2.0-flash` Vision으로 실제 추출.
키 없으면 데모 모드(샘플로 인식).

## 실데이터 교체 지점
- `data.js > PEER` : 데모 분포 → data.go.kr 한전 **지역별 용도별 전력사용량(15031941)** 집계로 대체
- `data.js > TARIFF` : 한전 2024 요금표 대표값(이미 실측 기반)
